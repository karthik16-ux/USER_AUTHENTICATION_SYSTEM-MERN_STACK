# MERN Stack User Authentication App

A full-stack web application built using the MERN stack that allows users to sign in using their name and email, with data securely stored in MongoDB.

---

# Features

-  User Sign-In with Name & Email
-  Data stored in MongoDB database
-  Fast and responsive UI
-  REST API integration
-  Full-stack implementation (Frontend + Backend)

---

# Tech Stack

Frontend

- React.js
- Axios
- CSS / Bootstrap

Backend

- Node.js
- Express.js

Database

- MongoDB

# API Endpoints

Method| Endpoint| Description
POST| /api/users| Save user details
GET| /api/users| Fetch all users

---

# How It Works

- User enters name and email on the frontend
- Data is sent to backend via API
- Backend processes request using Express
- Data is stored in MongoDB using Mongoose
- Response is sent back to frontend
